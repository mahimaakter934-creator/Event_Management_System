# Event_Management_System

This project is a web-based Event Management System developed using PHP, MySQL, HTML, CSS, and JavaScript. The system allows Event Managers to create, update, and delete events, while users can register, log in, join events, and view their joined events.

# Features
- User Registration & Login
- Role-based Access Control
- Add, Update, Delete, and Manage Events
- Event Joining System
- Participant Management
- Basic Form Validation

# Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- phpMyAdmin
- XAMPP

# Database
The project uses a MySQL database named `event_db` with the following tables:
- users
- events
- registrations

# Project Purpose
The purpose of this project is to provide a simple and user-friendly platform for managing events and participant registrations.
